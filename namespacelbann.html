LBANN optimizer base class Implements checkpoint and restart

Stochastic gradient descent with Adagrad. lr: float >= 0. Learning rate. epsilon: float >= 0.

Inspired by Kera.io implementation Stochastic gradient descent with RMSprop. lr: float >= 0. Learning rate. rho: float >= 0. epsilon: float >= 0. Fuzz factor.

Inspired by Kera.io implementation Stochastic gradient descent, with support for momentum, decay, and Nesterov momentum. lr: float >= 0. Learning rate. momentum: float >= 0. Parameter updates momentum. decay: float >= 0. Learning rate decay over each update. nesterov: boolean. Whether to apply Nesterov momentum.
